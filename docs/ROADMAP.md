- Set up a clean repository
- Remove tutorial artifacts
- Rebrand the product as DevInk
- Document the current baseline

- Replace remaining generic copy and branding
- Redesign the landing page and author experience

- Add ownership-aware blog CRUD

- Tie comments to real users

## Phase 4

- Add validation and sanitization
- Add tests for critical API flows
- Prepare deployment and screenshots
- Finalize portfolio-ready documentation
